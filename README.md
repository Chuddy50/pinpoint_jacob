# PinPoint

## how to run everything 

first to run the backend start pinpoint 
cd backend
pip install -r requirements.txt
uvicorn main:app --reload --host 127.0.0.1 --port 8000

then to run frontend starting at pinpoint 
cd Frontend
npm install
npm run dev


from there to access the  website, you go to http://localhost:5173
